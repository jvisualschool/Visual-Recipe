# 🍳 CHEF 비주얼 레시피 생성기 - 사용자 가이드

> AI가 만드는 나만의 비주얼 레시피북

---

## 📌 서비스 소개

**CHEF 비주얼 레시피 생성기**는 요리 이름만 입력하면 AI가 자동으로:
- 📝 레시피 (재료 + 조리 순서) 생성
- 🎨 아름다운 비주얼 레시피 이미지 생성

을 해주는 서비스입니다.

---

## 🚀 시작하기

### 접속 주소
```
https://jvibeschool.org/CHEF/
```

### 로그인
- **Google 계정**으로 로그인 (우측 상단)
- 로그인 후 **일일 3회** 레시피 생성 가능

---

## 🎯 레시피 생성 방법

### Step 1: 요리 이름 입력
```
예: 김치찌개, 파스타, 떡볶이, Carbonara...
```

### Step 2: 옵션 선택

#### 📐 화면 비율
| 옵션 | 설명 | 추천 용도 |
|------|------|----------|
| **세로형 (9:16)** | 스마트폰에 최적화 | 인스타 스토리, 핀터레스트 |
| **가로형 (16:9)** | 와이드 비율 | 유튜브 썸네일, 블로그 |
| **정방형 (1:1)** | 정사각형 | 인스타그램 피드 |

#### 🎨 비주얼 스타일 (7종)
| 스타일 | 특징 |
|--------|------|
| **모던 미니멀** | 깔끔하고 세련된 미니멀 디자인 |
| **3D 인포그래픽** | 3D 일러스트와 아이콘 활용 |
| **감성 수채화** | 부드러운 수채화 질감과 파스텔 컬러 |
| **그래픽 레코딩** | 손그림 마커 스타일 |
| **연필 스케치** | 연필로 스케치한 담백한 느낌 |
| **여중생 색연필** | 소녀 감성 색연필 일러스트 |
| **보태니컬 아트** | 식물도감 스타일의 섬세한 일러스트 |

#### 📋 구조/뷰 (레이아웃)
| 옵션 | 설명 |
|------|------|
| **기본 3단 구성** | 제목 → 재료 → 조리 순서 (세로 배치) |
| **벤토 그리드** | 도시락처럼 칸칸이 나뉜 그리드 |
| **원형 포커스** | 중앙에 메인 요리, 주변에 재료 배치 |
| **매거진 커버** | 잡지 표지 스타일 |

#### 💬 생성 언어
| 옵션 | 설명 |
|------|------|
| **한국어** | 한국어로만 생성 |
| **English** | 영어로만 생성 |
| **한/영 병기** | 한국어 + 영어 동시 표기 |

#### 🖼️ 텍스트 모드
| 옵션 | 설명 | 상태 |
|------|------|------|
| **이미지 내장 (AI)** | AI가 텍스트까지 직접 그림 | ✅ 사용 가능 |
| **HTML 오버레이** | 깔끔한 고해상도 텍스트 | 🚧 개발중 |

### Step 3: 생성 버튼 클릭
- "Generate Visual Recipe" 버튼 클릭
- 약 **30~40초** 소요
- 생성 중 재미있는 요리 상식 50가지가 랜덤으로 표시됨!

---

## 📚 갤러리 사용법

### 갤러리 접속
- 상단 메뉴에서 **"갤러리"** 클릭
- 또는 직접 접속: `https://jvibeschool.org/CHEF/gallery`

### 레시피 상세 보기
- 레시피 카드 클릭 → 상세 정보 확인
- **재료 목록** 복사 가능
- **조리 순서** 확인

### 이미지 다운로드
- 상세 보기에서 **다운로드 버튼** 클릭
- PNG 형식으로 고화질 저장

### 레시피 인쇄
- **인쇄 버튼** 클릭 → 브라우저 인쇄 기능 활용
- A4 용지에 최적화된 레이아웃

## ❓ 자주 묻는 질문 (FAQ)

### Q: 하루에 몇 개까지 만들 수 있나요?
**A:** 로그인 사용자 기준 **일일 3회**입니다.

### Q: 생성된 이미지 저작권은?
**A:** 생성된 이미지는 자유롭게 사용 가능합니다.

### Q: 이미지가 마음에 안 들면?
**A:** 같은 요리로 다시 생성하면 매번 다른 결과가 나옵니다.

### Q: 외국 요리도 되나요?
**A:** 네! 어떤 요리든 가능합니다. (예: Carbonara, Pad Thai, Sushi 등)

### Q: 생성 시간이 너무 오래 걸려요
**A:** 정상입니다. AI 이미지 생성에 약 30~40초가 소요됩니다.
